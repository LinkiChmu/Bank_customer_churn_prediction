## Кейс 1
## Бинарная классификация с использованием набора данных об оттоке банковских средств (Предсказание оттока клиентов банка)

Главная задача кейса предсказать, продолжит ли клиент пользоваться своим аккаунтом или закроет его

#### [Ссылка на Kaggle](https://www.kaggle.com/competitions/playground-series-s4e1/data)

### Основной функционал
*  Очистка и предобработка данных (обнаружение пропусков, нормализация, кодирование категориальных переменных)
*  Инжиниринг признаков (выделение важных характеристик клиента)
*  Обучение моделей: логистическая регрессия, деревья решений, случайный лес
*  Оценка качества моделей (метрики: precision, recall, F1-score, AUC-ROC)
